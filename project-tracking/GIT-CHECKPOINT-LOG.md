## 🔧 Git 작업 명령어 기록

### 저장소 초기화
```bash
git init                                    # Git 저장소 초기화
git add .                                   # 모든 파일 스테이징
git commit -m "체크포인트 메시지"          # 커밋 생성
```

### 향후 GitHub 연동 예정 명령어
```bash
git remote add origin [저장소URL]          # GitHub 저장소 연결
git branch -M main                          # 기본 브랜치명 변경
git push -u origin main                     # 첫 푸시
```
