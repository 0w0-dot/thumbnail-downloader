## 🚨 주의사항 및 팁

### 절대 하지 말 것
- ❌ private-info/ 폴더를 GitHub에 업로드
- ❌ API 키를 코드에 직접 입력
- ❌ 에러 메시지 무시하고 넘어가기

### 반드시 할 것
- ✅ 매일 GitHub에 코드 백업
- ✅ 에러 발생 시 즉시 SuperClaude에게 질문
- ✅ 작업 전 폴더 위치 확인
- ✅ 기능 완성 시마다 테스트

### SuperClaude와 효율적으로 작업하는 법
1. **구체적인 요청**: "버튼에 호버 효과 추가해줘" ✅
2. **모호한 요청 피하기**: "예쁘게 만들어줘" ❌
3. **파일 위치 명시**: 항상 정확한 경로 제공
4. **에러 전체 공유**: 에러 메시지 생략하지 말기
5. **결과 확인**: 작동 여부 즉시 피드백
